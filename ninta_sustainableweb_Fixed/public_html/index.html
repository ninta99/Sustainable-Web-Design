Facts about Front-End Web Design & Development

Approximately, 2-4% percent of global GHG emissions are digital activities involving data centers, transmission, and manufacturing which represents its importance in our society and everyday living (Lecorney, et. al, 2024).

University of Bristol’s study shows that streaming videos from YouTube by the hour produces approximately 10kg of CO2 (Tom Greenwood, 2020). In parallel, with the increase of digital devices reaching 30 billion now, web page size has increased by over 70% on desktop and 140% on mobile since 2015, these things show how ICT is becoming ubiquitous and easily accessible (Review of Web Analytic Tools for Sustainable Design, Lecorney et. al, 2024). The primary factors that could be used as indicators of carbon emissions include data transfer and carbon intensity of electricity. This leads to the constant availability and expectations regarding web design and development that seem to rely solely on aesthetics without thinking about the effect on the environment.  The data that we have and produce is a complex sociotechnical system made, in which it is intricately created for us to act and consume mindlessly, get more addicted, and be desensitized of the abundance and unregulated data which shows how much data is taking over that sociotechnical system in desensitizing us on its availability and constructing the endless need of it (Bodelier et. al, 2024).

As an individual, I want others to also explore and learn the concept of sustainable web design and development, especially for beginners and people who want to create straightforward, informative, and sustainable websites. The bloated design, unnecessary animations, and inefficient code, most times are unnecessary and thus also lead to the thought of how accessible and reliable is our website when the main purpose of it is to share information.

Why Is It Important?

What is sustainable web design and development? Referring to all the readings I have done, sustainable web design and development can be defined as an approach to design web services from front-end to back-end that takes the health of our planet into consideration. From the front-end and back-end development, we have to critically think about the importance of utilizing resources that reduces carbon emissions and energy consumption whilst building a website (Greenwood, 2020).

In this guideline specifically focuses in the front-end development of sustainable design development of a static website, and how we can manage to build something efficient when we’re trying to create a website.

File Guidelines (Tom Greenwood, LowTech Magazine, and SolarProtocol)

First, Download any code editor (I am using VScode), and then download the HTML, CSS, JavaScript Extensions on VScode. Create your work file on your desktop section, preferably a place that has direct access like the desktop or document file, with a clear label.

The next step, is to create file sections in the file consisting of public_html and _work. In the public_html, make 2 files consisting of css and img, and the index.html is not placed inside of a file doc. After creating these files, draft out your content and information on a .txt file or Word document. Gather images that you would want to use on that website and place them in the img file inside of public_html.

Design Guidelines for Front-End Development

Simplifying User Experience

Simplifying UX means that creating a user journey without having to load unnecessary data to open new pages and avoiding complicated directories. This can be achieved by streamlining the user journey without having to load unnecessary data, reducing pages on a website, avoiding yo-yo user journey, and ethically thinking about the user needs and navigation.

  • (i)With simple websites, this can be seen by having the navigation and all of the content on the same page, with separate section for the navigation. This is achieving on having just a page to load all of the data without creating several pages for one website, and organizing the content by creating same page navigation/directories.
  • (ii)This can also mean having straightforward information architecture with content, without extra features. If it’s a portfolio, you can show all the content directories on the same page without making the user have to search or find them by name.

24-Bit PNG Conversion

A 24-bit PNG file (which uses 16 million color channels) can also be compressed and converted to a PNG-8 file (uses a few block colors and eliminates unused colors), which reduces the size using minimal colors (best used for simple images that don’t need to have extensive and detailed colors).

Blurring Images

This part of design will help decrease the image file size and reduce the amount of variation between each pixel. Which will transfer less data. The main subject doesn't need to be blurred; additional background or edges can be blurred to reduce file size.

Tool used Pixlr.com

File Size Decreased by: 41.3%

Dithering Images

Dithering images rather than using fewer images can reduce a website's size when visuals are crucial in providing information. Dithering means that the object uses a limited color palette to create the illusion of depth, dithering uses adjacent pixels of different colors to give the appearance of a third color. his method is very helpful in preserving the image's depth and form while using fewer color pixels. However, the gradient of colors doesn’t appear smoothly since dithering will only use colors between the range of shades chosen.

Converted with Ditherit.com

File Size Decreased: 12.5%

File Size Decreased: 34.6%

Vector Imagery

Using SVG vector files to illustrate images or ideas can also decrease the file size used for an image. This is because vectors usually have minimal color usage, the unnecessary layers can also be removed and optimized.

Monochrome Images

As a general rule, the more colors used in an image or website, the larger the file will be. Reducing the variation of colors in an image or website can help curb energy consumption each time a website loads.Reducing file sizes saves energy throughout the whole network, from the data center through the transmission network to the devices of all users. Even though monochrome mode will use a lot of white, which is a bright pixel, the screen will also contain a lot of black used in a monochrome picture that will balance the usage of energy.

File Size Decreased: 36.5%

Using Effective Colors

Using Effective Colors to Decrease Energy Use First CRT Monitors used dark colors for websites to save energy. Now, in the usage of OLED and LCD, each pixel is illuminated individually, as a tiny LED still decreases energy usage and shows a significant difference with brighter color screens (ZDnet.com, Tung, 2018). Today, in the market based on LG, documented in YouTube channel HDTVTest, now OLED reached 22% of the PC gaming monitor market. However, this shows that most people still use LCD monitor because OLED has just been getting popular.

  • Color usage that can be followed consists of, (a) Using darker colors, or Black, which is most efficient color because pixels on the screen are switched off (OLED Screens). (b) Using less Blue, Blue pixels consume 25% more energy than green or red due to higher frequency of the color

Using Efficient Web Typography

To develop and design a sustainable website, we also have to be mindful of the typefaces used on a website. Commercial and open-source web fonts usually take a heavy toll on the carbon footprint by increasing data transfer and server requests to load a page. Using default typefaces that are preinstalled on a computer will be the best option, the typefaces can vary based on the computer system but it generally consists of selections of serif, sans serif, cursive, fantasy, system-ui, and monospace that can be chosen based on thetype of information to be represented.
